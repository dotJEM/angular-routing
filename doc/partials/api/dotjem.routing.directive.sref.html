sref (directive in module dotjem.routing )

Description

Provides a link to a state.

Usage

as attribute
<ANY sref
     params="{string}"
     activeClass="{string}">
   ...
</ANY>
as class
<ANY class="sref params: {string}; activeClass: {string};">
   ...
</ANY>

Parameters

  • params – {string} –

    Parameters for the state link.

  • activeClass – {string} –

    Class to add when the state targeted is active.
