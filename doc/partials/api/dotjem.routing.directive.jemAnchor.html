jemAnchor (directive in module dotjem.routing )

Description

Provides an anchor point for the $scroll service to use.

Usage

as element:
<jem-anchor
       id="{string}">
</jem-anchor>
as attribute
<ANY jem-anchor="{string}">
   ...
</ANY>
as class
<ANY class="jem-anchor: {string};">
   ...
</ANY>

Parameters

  • jemAnchor|id – {string} –

    Identifier of the anchor
