id (directive in module dotjem.routing )

Description

Provides an anchor point for the $scroll service to use.

Usage

as attribute
<ANY id
     jemAnchor="{string}">
   ...
</ANY>
as class
<ANY class="id jemAnchor: {string};">
   ...
</ANY>

Parameters

  • jemAnchor|id – {string} –

    Identifier of the anchor
