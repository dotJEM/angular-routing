$resolve (service in module dotjem.routing )

Description

The $resolve service is used to resolve values for states

Dependencies

  • $q
  • $injector

Methods

  • clear(key)

    Clears a value in the resolver.

    Parameters

    • key – {string} –

      A key for the value to remove from the resolvers cache.

  • clear(keys)

    Clears a list of values in the resolver.

    Parameters

    • keys – {Array} –

      An array of keys for the values to remove from the resolvers cache.

  • push(key, value)

    Pushes a value into the resolver.

    Parameters

    • key – {string} –

      A key for the value.

    • value – {Object} –

      The value.
