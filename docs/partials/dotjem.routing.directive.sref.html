sref
directive in module dotjem.routing

Description

Provides a link to a state.

Usage

as attribute
<ANY sref
     params="{string}"
     active-class="{string}">
   ...
</ANY>
as class
<ANY class="sref params: {string}; active-class: {string};">
   ...
</ANY>

Parameters

ParamTypeDetails
paramsstring

Parameters for the state link.

activeClassstring

Class to add when the state targeted is active.
